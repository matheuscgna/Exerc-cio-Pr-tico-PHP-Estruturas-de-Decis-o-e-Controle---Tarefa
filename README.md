# Exerc-cio-Pr-tico-PHP-Estruturas-de-Decis-o-e-Controle---Tarefa
10 atividades de PHP, proposta pelo Prof. Wendell Bento Geraldes - IFG.

Estruturas Condicionais (IF, ELSE, SWITCH)
1.Par ou Ímpar: Crie uma variável $numero. Use if e else para exibir se o número é "Par"
ou "Ímpar".
2.Aprovação Escolar: Crie três variáveis para notas ($n1, $n2, $n3). Calcule a média. Se
a média for 7 ou mais, exiba "Aprovado"; entre 5 e 6.9, "Recuperação"; abaixo de 5,
"Reprovado".
3.Calculadora de IMC: Com base em uma variável $peso e $altura, calcule o IMC (peso /
altura²). Use condicionais para dizer se a pessoa está abaixo do peso, no peso ideal ou
acima do peso.
4.Menu de Cores (Switch): Crie uma variável $cor. Use switch para exibir uma mensagem
diferente para "azul", "vermelho" e "verde". Se não for nenhuma dessas, exiba "Cor
desconhecida".
5.Verificador de Idade: Crie uma variável $anoNascimento. Calcule a idade atual. Exiba se
a pessoa é maior de idade ou menor, e se o voto é obrigatório (entre 18 e 70 anos).
Estruturas de Repetição (FOR, WHILE, FOREACH)
6.Tabuada: Escolha um número e use o laço for para exibir a tabuada dele de 1 a 10.
7.Contagem Regressiva: Use um laço while para exibir uma contagem regressiva de 20
até 0, pulando de 2 em 2.
8.Soma de Intervalo: Use um laço para somar todos os números de 1 a 100 e exiba o
resultado final.
9.Lista de Frutas (Foreach): Crie um array com 5 nomes de frutas. Use o foreach para
listar cada fruta em uma linha diferente dentro de uma lista HTML (<li>).
10.Filtrando Lista: Crie um array de números. Use foreach e if para exibir apenas os
números que são maiores que 10.
